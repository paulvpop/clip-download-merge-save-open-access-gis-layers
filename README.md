This repository will help a user to clip-and-download (using google Earth Engine) or download-and-clip (using R) large open access rasters. This makes the downloads smaller or the data smaller after download, so that unnecessary download and/or storage can be avoided. This will also help in loading the layers in QGIS without it crashing or hanging. Once loaded in QGIS, they can further be merged, clipped to desired area, and then saved.
